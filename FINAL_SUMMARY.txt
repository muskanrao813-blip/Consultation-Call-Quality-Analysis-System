                      OPTIONAL: FULL PROCESSING SETUP
================================================================================

To enable audio transcription & AI analysis:

1. GET GOOGLE CLOUD CREDENTIALS (30 minutes)
   - Service account JSON key (for Speech-to-Text)
   - Gemini API key (for AI analysis)
   - GCS bucket (for audio storage)

2. ADD TO .env FILE
   GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json
   GEMINI_API_KEY=your-api-key
   GCS_BUCKET_NAME=dietician-qa-audio

3. RESTART SERVER
   python -m uvicorn app.main:app --reload --port 8001

4. PROCESSING WILL AUTO-START
