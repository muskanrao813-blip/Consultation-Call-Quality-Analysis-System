   - Go to: https://ai.google.dev/
   - Create API key
   - Set in .env: GEMINI_API_KEY=your-key-here

WITHOUT CREDENTIALS:
  - Portal still works
  - File upload still works
  - Database stores data correctly
  - Processing will fail with error message

================================================================================
                            TROUBLESHOOTING
================================================================================

ISSUE: "Port 8001 already in use"
SOLUTION: Use different port:
  python -m uvicorn app.main:app --port 8002

ISSUE: "Module not found" errors
SOLUTION: Reinstall dependencies:
  pip install -r requirements-minimal.txt

ISSUE: Database locked
SOLUTION: Delete database and reinitialize:
  del test.db
  python init_db.py

ISSUE: Server won't start
SOLUTION: Check Python is installed:
  python --version

  If not, reinstall from: https://www.python.org/downloads/
