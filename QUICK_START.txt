  3. Run installer
  4. Restart computer

================================================================================
                          TROUBLESHOOTING
================================================================================

"Can't open http://localhost:8001"
  → Server not running
  → Open PowerShell:
     cd C:\Users\muskan.rao\Documents\claude\dietician-qa
     python -m uvicorn app.main:app --reload --port 8001

"Port 8001 already in use"
  → Use different port:
     python -m uvicorn app.main:app --reload --port 8002
     Then open: http://localhost:8002

"File won't upload"
  → Check file has 3 required columns
  → Check file is valid Excel/CSV
  → Check server is running
  → Open browser console (F12) for errors

"Ollama connection refused"
  → Ollama server not running
  → Open PowerShell window and run: ollama serve
  → Keep window open while processing
