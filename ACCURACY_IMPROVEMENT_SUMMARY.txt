5. SPECTRAL ENHANCEMENT (Before Transcription)
   ──────────────────────────────────────────────────────────
   Current: Basic noise gate + bandpass filter
   Upgrade: Add spectral subtraction + Wiener filtering

   Advanced Techniques:
     • Spectral subtraction (aggressive noise reduction)
     • Wiener filter (preserve speech, suppress noise)
     • Voice activity detection (isolate speech frames)

   Expected Gain: +3-5% accuracy
   Effort: 2-3 hours

   Status: NOT IMPLEMENTED

6. CONTEXT-AWARE ENTITY EXTRACTION
   ──────────────────────────────────────────────────────────
   Current: Pattern matching + Claude extraction
   Upgrade: Use Claude to extract relationships + context

   Extract Not Just:
     ❌ "Bajaj Finserv Health"

   But Also:
     ✅ "Bajaj Finserv Health (organization) + consultation (service type)"
     ✅ Patient health status, appointment details, follow-ups

   Expected Gain: +10-15% entity accuracy
   Effort: 2-3 hours

   Status: NOT IMPLEMENTED

7. MULTI-MODEL ENSEMBLE
   ──────────────────────────────────────────────────────────
   Current: Single Groq transcription
   Idea: Get transcriptions from 2-3 models, vote on words

   Models:
     • Groq/Whisper (current)
     • Google Cloud Speech-to-Text (Hinglish support)
     • OpenAI Whisper API (highest quality)

   Result: Final transcript = majority vote on each word

   Expected Gain: +5-10% accuracy
   Cost: 2-3x API costs (Groq usage increases)
   Effort: 4-6 hours

   Status: NOT IMPLEMENTED

   ──────────────────────────────────────────────────────────

   Expected Gain: +5-8% accuracy
   Effort: 4-6 hours
